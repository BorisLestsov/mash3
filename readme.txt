Операционная система: Ubuntu

Инструкция по сборке:
1) Запустить make в папке проекта
2) Исполняемый файл - mash3 в папке bin. Запускать необходимо именно из папки bin.


Описание выполенной работы:

Базовая часть:
Травинки рисуются инстансингом. Все травинки имеют разную высоту, разную толщину, разный цвет и разный поворот. Колеблются также все по разному.

Кроме базовой части, реализованы:

1) MSAA. Отключается/включается латинской клавишей 'a'. (Хорошо виден на всех объектах, кроме бамбука и капусты, так как при их отрисовке используется директива discard, которая не обрабатывается MSAA)
2) Текстура на травинках
3) Затемнение травы ближе к земле для имитации эффекта Ambient Occlusion
4) Иммитация пожелтения травы (трава имеет разный цвет, некоторые травинки более желтые, чем остальные)
5) Более реалистичная симуляция на CPU (все травинки колеблются по разному, учитывается влияние гравитации)
6) Добавлены различные объекты:
	- Два различных вида растений (капуста и бамбук из minecraft (lol))
	- Добавлено статичное сухое дерево
	- Два вида животных (кролик и кот)
	- Камень
	- Трактор
7) Добавлен скайбокс
